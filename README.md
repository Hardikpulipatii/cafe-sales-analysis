# Retail Café Sales Analytics using Excel & SQL

## Tools Used
- Excel
- MySQL

## Project Overview
Performed end-to-end data analysis on café sales dataset including data cleaning, transformation, and SQL-based analysis.

## Steps Performed
- Cleaned messy data (handled missing values, UNKNOWN, ERROR)
- Imported dataset into MySQL
- Performed analysis using SQL (GROUP BY, aggregations)
- Applied advanced SQL (Window functions, CTEs)

## Key Insights
- Coffee is the top-selling product
- Revenue is evenly distributed across payment methods
- Sales show fluctuations with peak days
- Unknown data handled for better analysis
